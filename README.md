# Library Management System (Java)

## 🚀 Features
- Add books
- Search books
- Issue books
- Return books
- File handling (data persistence)
- GUI using Java Swing

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- File Handling
- OOP Concepts

## ▶️ How to Run
1. Compile: javac *.java
2. Run: java LibraryGUI

## 📌 Description
This is a file-based Library Management System built using Java. It allows users to manage books, issue and return them, and store data permanently using file handling.
